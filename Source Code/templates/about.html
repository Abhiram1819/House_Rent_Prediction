About  Predict.com

House Rent Predictor is a simple tool to calculate the House Rents in various areas of the city. This tool would benefit the residents of hyderabad who are looking to rent a house. This tool is made using Machine Learning Model. Flask is used as backend and HTML,CSS,JavaScript for frontend. The data of hyderabad house rents is collected from kaggle. The raw data collected is pre-processed. We implemented Linear Regression and Random Forest Regressor on the data. Best accuracy is obtained using Linear Regression. So, we used Linear Regression Model to predict the House Rents.
